Feather
========================

A Simple, Flexible and Beautiful torrent webapp

Installation ( BETA )
========================

Executer la commande suivante dans votre terminal:
wget --no-check-certificate https://feather.vigitas.com/dl/feather && chmod 0777 feather && ./feather install
